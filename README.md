# W8(Weight)  
### W8 is a simple weight management bot(Kakao talk, LINE, Facebook Messenger)

## 친구추가(서비스종료)
* [KakaoTalk](https://pf.kakao.com/_bknhxl)
* [LINE](https://line.me/R/ti/p/%40vbw9172l)
* Facebook은 비공개 운영 중

## 주요기능
* 몸무게 입력
* 목표 설정
* 체중 변화량 확인(차트, 테이블)
* 몸무게 입력 푸시알람(LINE, Facebook)

## 기술스펙
* Node.js(Express)
* Mysql
* EJS
* Chart.js
* Bootstrap
* node-cron
